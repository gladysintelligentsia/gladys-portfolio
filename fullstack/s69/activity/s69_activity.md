# Session 69 - Application Debugging


## Activity


#### Activity Instructions  

1. Inside your **individual** folder, create an **s69** folder, then create a new folder named **activity** inside it.

2. Pull the **client** and **server** folders from the **batch repository**, then **move** both folders into the **activity** folder.

3. **Analyze and debug the Full Stack Games Library App** to avoid any errors

	- Ensure that all **pages works as intended.**
	- Ensure that all **routes and controllers are working properly**
	- **Rename or refactor files and components** as needed
		- **Note that the app is a Games Library**
	- **Delete unnecessary files** as needed.

4. Update the **individual repo** with the **appropriate commit message.**

5. Link your individual repo to Boodle as s69.
